# VQA Annotation System

This is a VAQ annotation system used for V2X roadside data labeling.

## Install

- make sure you have install node.js v20+

```sh
# install pnpm
npm i pnpm -g

# enable corepack
corepack enable

# install packages
pnpm i
```

## Architecture

### Front-end

Vue3 + vite

### Back-end

express
